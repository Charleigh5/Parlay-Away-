## Testing

This project uses Vitest for unit testing. See [TEST_README.md](./TEST_README.md) for detailed testing documentation.

### Quick Start

```bash
# Run all tests
npm test

# Run tests in watch mode
npm test -- --watch

# Run tests with coverage
npm run test:coverage

# Run tests with UI
npm run test:ui
```
